Znacznik tabelki <table>

Służy do utworzenia tabeli składającej się z komórek utworzonych za pomocą znaczników <td> oraz <th> zgrupowanych w wiersze dzięki znacznikowi <tr>. Komórki tabeli <td> znajdują się wewnątrz znaczników wierszy <tr>. Może zawierać część nagłówkową <thead>, ciało <tbody> i stopkę <tfoot>.

<th> - znacznik nagłówka kolumny w tabelce, zawiera domyślnie pogrubiony i wyśrodkowany tekst

<td> - znacznik standardowej pojedynczej komórki

<tr> - znacznik wiersza

PRZYKŁAD


<p>Ranking FIVB</p>
  <table>
    <thead>
      <tr><th>Miejsce</th><th>Reprezentacja</th><th>Punkty</th></tr>
    </thead>
    <tbody>
       <tr><td>1</td><td>Brazylia</td><td>315</td></tr>
        <tr><td>2</td><td>Stany Zjednoczone</td><td> 270 </td></tr>
        <tr><td>3</td><td>Polska</td><td>262</td></tr> 
        <tr><td>4</td><td>Włochy</td><td>240</td></tr>
        <tr><td>4</td><td>Rosja</td><td>240</td></tr>
    </tbody>
    <tfoot>
      <tr><td>ostatnie</td><td>Kamerun</td><td>27</td></tr>
    </tfoot>
  </table>

Znacznik listy nienumerowanej <ul>

Służy do wskazania listy nieuporządkowanej, czyli zestawu punktów, które nie mają porządku liczbowego i ich kolejność nie ma znaczenia. Poszczególne punkty w liście są definiowane poprzez element <li>, który jest jedynym dopuszczalnym elementem w znaczniku <ul>. Listy można w sobie zagnieżdżać.

<li> - pojedyncza pozycja w liście

PRZYKŁAD


     <ul>
      <li>pierwsza pozycja</li>
        <li>druga pozycja  
          <ul>
            <li>druga pozycja pierwszy podpunkt</li>
            <li>druga pozycja drugi podpunkt</li>
            <li>druga pozycja trzeci podpunkt</li>
          </ul>
        </li>   
      <li>trzecia pozycja</li>
    </ul>

Znacznik listy numerowanej <ol>

Służy do wskazania listy uporządkowanej, czyli zestawu punktów, które mają porządek liczbowego, a ich kolejność ma znaczenie. Poszczególne punkty w liście są definiowane poprzez element <li>, który jest jedynym dopuszczalnym elementem w znaczniku <ol>. Listy można w sobie zagnieżdżać.

<li> - pojedyncza pozycja w liście

PRZYKŁAD


    <ol>
      <li>pierwsza pozycja</li>
        <li>druga pozycja  
          <ol>
            <li>druga pozycja pierwszy podpunkt</li>
            <li>druga pozycja drugi podpunkt</li>
            <li>druga pozycja trzeci podpunkt</li>
          </ol>
        </li>   
      <li>trzecia pozycja</li>
    </ol>

Znacznik tekstu formatowanego <pre>

Reprezentuje zawartość, której treść ze względu na swoje pochodzenie oraz formę musi zostać zaprezentowana w sposób z góry narzucony przez różne standardy typografii. Zawartością elementu <pre> może być na przykład fragment kodu komputerowego zamkniętego w znaczniki <code> </code>.

<code> - blok kodu

PRZYKŁAD


    <pre><code>
    #include <iostream>
    using namespace std;

    int main()
    {
        cout << "Hello World!";

        return 0;
    }

    </code></pre>
